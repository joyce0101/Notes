# 冒泡排序
## 一、算法基本思想  
  1、从无序队列的头部开始，两两比较，根据大小交换位置，最后将最大的元素交换到无序队列的队尾，成为有序队列的一部分，下次继续这个过程。时间复杂度为O（n2）。

## 二、具体实现  
<pre>  
void pop_sort(int a[],int len)
{
    int i,j;
    for(int i=0;i<len;i++)
    {
        for(int j=0;j<len-i-1;i++)
        {
            if(a[j]>a[j+1])
                swap(a[j],a[j+1]);
        }
    }
}
